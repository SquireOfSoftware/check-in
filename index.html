README

To use this system you can access:

  • Scanner

    This is for when you want to scan people in.

    Please take note that the camera relies on the web certificates.
    As such you will need to install the self signed certificate here: Download certificate

    You will need to follow some instructions to install the certificate, once installed, you will need to allow for the browser to let you visit the page despite the security warning.

    You may also need the web certificate for the domain.
    As such you may need to install the certificate for the application: Download certificate

    Once the certificates are installed, try visiting this link: "secure" scanner link

  • Report

    This is for when you want to look at the report of signed in people or if you want to manually sign someone in.
